Welkom bij mijn website

Princess Asheshe

Ik ben princess Asheshe. Ik ben 18 jaar oud en zit op piustien in 5BIN. Op deze website zal u al mijn progressie zien van het afgelopen jaar en wat ik in de toekomst ook nog zal doen. Alles op deze website is door mij persoonlijk gemaakt (met heel wat hulp van mijn leraar). Ik heb er veel tijd en moeite in gestopt. Tijdens het maken van deze website ben ik door veel struikel punten gekomen en heb ik ze best wel vaak genegeerd tot ik het dan wel moest fiksen anders kon ik neit verder het lukte niet altijd maar wel vaak dus ik ben er heel erg trots op. Het moeilijkste aan een website is niet de css of de html het is eerder tijd. de tijd die je nodig hebt om een website te maken moet je nemen. je moet er veel over na denken en je ontwerp kan een paar keer veranderd worden maar als je genoeg tijd neemt dan zal het absolute lukken. Hopelijk geniet u van het doorgaan van mijn website.

interesses

mijn interesses hebben veel te maken met mijn hobby's. Op netflix heb je best wel veel anime's en dat is eigenlijk wat ik het meest kijk op netflix. Ik kijk ook anime's op gogoanime maar niet zo veel als op netflix want de anime op netflix is meer mijn stijl en gogoanime is een illegale site dus ik krijg best veel virusen als ik op dat ga kijken. Ik zit veel op mijn gsm omdat ik heel veel manga lees. Soms lees ik manga omdat anime altijd heel lang erover doet om een nieuw seizoen uit te brengen (behalve pokemon, one piece en Naruto/ Boruto die zijn redelijk snel.) dus als ik echt niet kan wachten lees ik het vervolg van de anime in de manga want elke anime heeft een manga. Anime's worden pas gemaakt als je ziet dat de manga heel erg bekend is en een set aantal lezers heeft. Dus eerst schrijf je een manga en dan film je een anime. Ik lees ook soms de manga alleen en skip de anime omdat het een goede manga is maar het is van een chinees en een chineese anime laat altijd belangerijke dingens vallen omdat ze "geen tijd" hebben.
